# Netflix-Website-Clone

## 1. 프로젝트의 동기
- 클론 코딩을 통한 HTML 과 CSS 언어의 학습

## 2. 프로젝트 주요 내용
- 헤더섹션 좌측에는 넷플릭스 로고, 우측에는 언어선택 및 회원가입 버튼을 만들었습니다.
- 상단 중앙에는 환영 텍스트와 이메일 구독 양식을 만들었습니다.
- 하단 중앙에는 FAQ를 만들어 + 버튼 클릭 시 세부사항이 표시되게 만들었습니다.
